# Support

## Getting Help

**Found a bug?**
Open a [GitHub Issue](https://github.com/mcdow-webworks/ce-pipeline-testbed/issues). Include steps to reproduce, expected behavior, and actual behavior.

**Have a question or idea?**
Start a [GitHub Discussion](https://github.com/mcdow-webworks/ce-pipeline-testbed/discussions). Discussions are the right place for questions, feedback, and general conversation about the project.

## Note

This is a test environment for the CE pipeline — not a production repository. Response times may vary.
