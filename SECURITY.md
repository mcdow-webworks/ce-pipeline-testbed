# Security Policy

## Supported Versions

This is a test/sandbox repository. No production systems depend on this code.

## Reporting a Vulnerability

If you discover a security vulnerability in this repository, please report it by email:

**Contact:** mcdow@webworks.com

Include the following in your report:

- A description of the vulnerability
- Steps to reproduce
- Potential impact

We will acknowledge your report within 48 hours and provide an estimated resolution timeline. We ask that you give us reasonable time to address the issue before any public disclosure.

Do not open a public GitHub issue for security vulnerabilities.
